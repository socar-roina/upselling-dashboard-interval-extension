2. 정확한 수락 건수 (환불 제외)
WITH net_data AS ( SELECT DATE(created_at, "Asia/Seoul") AS dt, reservation_id, SUM(amount) AS net_amount FROM `socar-data.dbt.stg_tianjin_replica__paid_info` WHERE DATE(created_at, "Asia/Seoul") BETWEEN "2026-05-13" AND "2026-05-21" AND paid_type = 'upselling_discount' AND DATETIME(created_at, "Asia/Seoul") >= DATETIME("2026-05-13 14:09:30") GROUP BY dt, reservation_id ) SELECT dt, SUM(CASE WHEN net_amount > 0 THEN 1 ELSE 0 END) AS net_accepted_rsv, SUM(CASE WHEN net_amount > 0 THEN net_amount ELSE 0 END) AS company_burden_net, SUM(CASE WHEN net_amount > 0 THEN 4900 ELSE 0 END) AS revenue_increment FROM net_data GROUP BY dt ORDER BY dt

대시보드 용어 정리

옵션 화면 (업셀링 바텀시트)결제 직전에 화면 아래에서 올라오는 시간연장 옵션 카드가 뜬 화면. 사용자가 이 화면을 봐야만 시간연장이 가능.
시간연장 카드 누르기 (옵션 토글)옵션 화면에서 사용자가 "연장 시간으로 변경" 카드를 직접 눌러서 선택한 행위. 결제 의도 표시.
결제 버튼 누르기옵션 화면 맨 아래 결제 진행 버튼 누른 행위. 시간연장을 선택했든 안 했든 결제로 넘어가는 의미라 "수락" 과는 다름.
실제 수락한 예약의 결제 합계가 양수인 경우. 결제 후 환불되지 않고 끝까지 유지된 경우만 카운트.
매출 증분사용자가 시간연장 옵션 수락으로 추가로 낸 돈. 한 건당 4,900원 고정 × 수락 건수.
회사 할인 비용실제 연장 요금 - 사용자가 낸 4,900원. 차액은 회사가 할인 형태로 지원. 수락 건들의 차액 합계.
수락률옵션 화면을 본 회원 중 실제로 수락한 회원의 비율. 실제 수락 회원 수 ÷ 옵션 화면 본 회원 수.
실험군 / 대조군 (A/B)실험군(B군) = 옵션 화면을 보는 회원, 대조군(A군) = 옵션 화면이 안 뜨는 회원. 50:50으로 무작위 분배.
